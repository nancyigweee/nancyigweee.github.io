Design Principles

Repetition

Old Navy

oldnavy.com

The same elements are repeated more than once on this website and it makes it look more organized and uniform. Repetition is used throughout this website.

White Space and Clean Design

Apple

apple.com

There's a lot of negative space between the elements on the page. This makes the website look clean and uncluttered. Using white space also makes the text easier to read.

Visual Hierarchy

Nike

nike.com

The website uses visual hierarchy to attract the user to the "Go to Checkout" button first. The most important elements catch our eyes first and get more attention.
